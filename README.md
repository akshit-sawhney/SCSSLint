# SCSS Lint

A linting support for scss

## Getting Started

1. Run npm install
2. Run gulp scss-lint

### Prerequisites

1. Node
2. NPM

### Installing

A step by step guide to see this in action.

Install all the dependencies

```
npm install
```

Run the gulp task

```
gulp scss-lint
```
